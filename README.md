# Cobblemon Legacy Nova Downloads

Repositorio de downloads da nova base do launcher Cobblemon Legacy.

Este repositorio deve armazenar:

- `manifest.json`
- releases com mods/resourcepacks/configs
- arquivos auxiliares publicos do launcher

O launcher deve apontar para:

```text
https://raw.githubusercontent.com/gotardelo/cobblemonlegacy-nova-downloads/main/manifest.json
```

## Versoes travadas

- Minecraft: `1.21.1`
- Loader: `Fabric`
- Fabric Loader: `0.19.3`
